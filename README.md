# SIMS PPOB - Web Programmer (React JS) Technical Test

This project is a web-based Payment Point Online Bank (PPOB) application built as part of the Take Home Test for the **Web Programmer (React JS)** position. 

The application allows users to register, log in, view their profile and balance, perform top-ups, pay for various services, and view their transaction history.

## 🚀 Tech Stack

This project is built using modern frontend technologies to ensure performance, type safety, and maintainability:

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **UI Components:** [Shadcn UI](https://ui.shadcn.com/)
* **State Management:** [Redux Toolkit](https://redux-toolkit.js.org/) & [Redux Persist](https://github.com/rt2zz/redux-persist)
* **Data Fetching & Caching:** [TanStack Query (React Query)](https://tanstack.com/query/latest)
* **Form Handling & Validation:** [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
* **Icons:** [Lucide React](https://lucide.dev/)

## ✨ Key Features

1. **Authentication:** User Registration and Login with JWT implementation.
2. **Dashboard:** Displays user profile summary and current balance.
3. **Profile Management:** Users can view and update their profile picture and personal information.
4. **Top Up:** Users can input a specific amount or select from predefined nominals to add balance.
5. **Service Payment:** Interactive menu to select and pay for various PPOB services (e.g., PLN, Pulsa).
6. **Transaction History:** Displays a list of transactions with **Infinite Scrolling (Load More)** and filtering capabilities.
